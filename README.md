# The Light of Errors

## Usage

### installation

```bash
$ bundle install
```

### start the app

```bash
$ ./app.rb -e production
```

### Check

```bash
$ curl -XGET http://localhost:4567/
```

### Fire!

```bash
$ curl -XPOST http://localhost:4567/ -d ''
```

### Turn off

```bash
$ curl -XDELETE http://localhost:4567/
```
